# Тестовое задание в Перфоманс Лаб

## Постарался написать понятные докстринги к каждому заданию, также немного оптимизировать скорость выполнения при помощи локальных переменных, ну и максимально отловить всевозможные ошибки, если задания будут проверяться на автотестах.

- **Задание 1** — работа с круговыми массивами и построение пути по заданным параметрам.
- **Задание 2** — определение положения точек относительно эллипса.
- **Задание 3** — формирование отчёта на основе JSON-файлов с результатами тестов.
- **Задание 4** — вычисление минимального количества ходов для приведения всех элементов массива к одному числу.
